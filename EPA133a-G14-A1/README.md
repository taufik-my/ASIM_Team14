## â–¶ How to Run the Code

The **main notebook** is:

```

```

This notebook:

* Loads the raw road and bridge datasets
* Performs preprocessing and restructuring
* Applies data quality checks
* Detects and repairs spatial outliers
* Exports cleaned datasets to:

```
data/processed/
```

The other notebooks (`Chainage_monotony`, `Check_boundary_with_shp`, `Fixing_road`) were used during task division and intermediate development.

To reproduce the analysis:
